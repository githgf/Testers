策略模式
    定义：
        对一系列的算法定义，并将每一个算法封装起来，而且使它们还可以相互替换。策略模式让算法独立于使用它的客户而独立变化。
    优点
        有策略模式提供了管理相关的算法族的办法、策略模式提供了可以替换继承关系的办法、使用策略模式可以避免使用多重条件转移语句。
    总结
        涉及到三个角色：strategy抽象策略角色，ConcreteStrategy具体策略角色，context策略引用，
        和工厂对象相似，但是在context对象初始化时传入具体的策略对象，再调用其算法
